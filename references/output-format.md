# Output Format

Use this format for recommendations.

```md
## 推荐结论
优先使用：<candidate>

理由：<one short paragraph>

## 候选能力
1. <name>
   - 来源：本地索引 / 官方来源 / 当前可用 / 第三方 / 待验证
   - 适合：...
   - 不适合：...
   - 安装或授权成本：...
   - 风险：...

2. <name>
   - 来源：...
   - 适合：...
   - 不适合：...
   - 安装或授权成本：...
   - 风险：...

## 建议执行路径
1. ...
2. ...
3. ...

## 剩余不确定性
- ...
```

Keep the answer short unless the user asks for a full comparison.

When the user only needs a quick routing answer, use this shorter form:

```md
建议用：<candidate>

原因：...

备选：<candidate>, <candidate>
```
